What's Upstage?

Upstage lets you create HTML presentations.

<div class="slide">
  <h2>My Wonderful Title</h2>
  <p>Some points:</p>
  <ul>
    <li>Point 1</li>
    <li>Point 2</li>
    <li>Point 3</li>
  </ul>
</div>
var deck = new Y.Upstage({
    srcNode: "#bd"
});

deck.plug([
    Y.Plugin.UpstageKeyboard,
    Y.Plugin.UpstageBlank,
    Y.Plugin.UpstageGesture,
    Y.Plugin.UpstagePermalink
]);

deck.render();

Slides are plain HTML.

Upstage transforms your markup into a presentation, complete with transitions, touch and mouse gestures, and keyboard interaction.

Upstage itself is a tiny (3K) YUI 3 Widget, which relies on Widget plugins to provide its features, so you can use only what you need.

Getting Started

  1. Get the Upstage YUIConf 2011 code.
  2. Edit index.html and go!

Why HTML?

Presenting with HTML is not for everyone. There are some good reasons:

  • Syntax highlighting
  • Edit with your favorite editor
  • Version control with Git
  • Share on GitHub
  • Write in Markdown
  • Run JS in your deck
  • Share without SlideShare
  • Dogfooding YUI
  • Web tech > native apps

    if (this.attendees.indexOf("me") === -1) {
        throw new Error("Without me, it's just Aweso.");
    }
    return true;

var yuiconf = new Conference();
if (yuiconf.isAwesome()) deck.fire("warp", 1);

Upstage Plugins

The Upstage core, upstage-slideshow, provides the framework for the presentation.

Make it useful by plugging in some of these:

  • UpstageKeyboard: Keyboard navigation.
  • UpstageGesture: Gesture navigation.
  • UpstageBlanker: Try it out: press B to blank the show. Great for Q&A!
  • UpstagePermalink: Provides permalinks to slides that are unique per-slide.
